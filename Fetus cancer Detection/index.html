How It Works
🔬
ResNet-50 CNN
A 50-layer deep residual network pre-trained on ImageNet and fine-tuned on 2,966 labelled fetal ultrasound images spanning three diagnostic classes.
🧠
Class Balancing
Weighted random sampling and inverse-frequency loss weights ensure every class — despite large size differences — contributes equally to training.
🗺️
Grad-CAM Explainability
Gradient-weighted Class Activation Mapping highlights the exact regions of the scan the model focuses on — giving doctors visual insight into every decision.
